# Chosen 2026 — UI Kits

The Chosen brand outputs are primarily **collateral and content surfaces**, not a software product. Three kits live here:

- `social/` — Instagram square + story templates for promo, speaker reveals, scripture posts
- `email/` — HTML email templates (welcome, schedule reminder, partnership outreach)
- `print/` — flyer + poster recreations matching the supplied 2026 *Be Radiant* artwork

Open each kit's `index.html` to see the full set rendered. Components are factored as JSX inside each kit folder, designed to be remixed into real campaigns.
